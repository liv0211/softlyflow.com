## 🚀 如何开始使用

### 1. 本地运行网站

在终端中运行：

```bash
cd /Users/liang/Desktop/softlyflow.com
npm run dev
```

然后在浏览器打开：http://localhost:3000

### 2. 添加新文章

编辑文件：`lib/articles.ts`

找到 `articles` 数组，添加新文章：

```typescript
{
  slug: 'your-article-slug',           // URL 路径，使用英文和连字符
  title: 'Your Article Title',         // 文章标题
  excerpt: 'Brief description...',      // 简短描述，显示在列表页
  content: `# Your article content...`, // 文章内容，支持 Markdown
  date: '2024-01-20',                   // 发布日期，格式：YYYY-MM-DD
  category: 'Productivity',             // 分类
  tags: ['AI', 'productivity'],         // 标签数组
  readTime: 5,                          // 预计阅读时间（分钟）
  seoTitle: 'SEO Optimized Title',      // SEO 标题（可选）
  seoDescription: 'SEO description...', // SEO 描述（可选）
}
```

### 3. 修改网站信息

- **网站标题和描述**：编辑 `app/layout.tsx` 文件
- **导航菜单**：编辑 `components/Header.tsx` 文件
- **页脚信息**：编辑 `components/Footer.tsx` 文件
- **首页内容**：编辑 `app/page.tsx` 文件

### 4. 自定义样式

- **颜色主题**：编辑 `tailwind.config.ts` 文件中的 `primary` 颜色
- **字体**：在 `app/layout.tsx` 中修改字体链接
- **全局样式**：编辑 `app/globals.css` 文件

## 📝 Markdown 语法支持

文章内容支持以下 Markdown 语法：

- **标题**：`# 一级标题`、`## 二级标题`、`### 三级标题`
- **粗体**：`**粗体文字**`
- **斜体**：`*斜体文字*`
- **链接**：`[链接文字](https://example.com)`
- **列表**：
  ```
  - 列表项 1
  - 列表项 2
  ```

## 🌐 部署到线上

### 推荐方式：Vercel（最简单）

1. **注册账号**
   - 访问 https://vercel.com
   - 使用 GitHub 账号登录

2. **准备代码**
   - 将代码推送到 GitHub（如果还没有）
   - 在 GitHub 创建新仓库
   - 将本地代码推送到仓库

3. **部署**
   - 在 Vercel 点击 "New Project"
   - 选择你的 GitHub 仓库
   - Vercel 会自动检测 Next.js 项目并部署

4. **绑定域名**
   - 在 Vercel 项目设置中添加域名 `softlyflow.com`
   - 按照提示配置 DNS 记录

详细步骤请查看 `DEPLOYMENT.md` 文件。

## ❓ 常见问题

**Q: 如何修改网站颜色？**
A: 编辑 `tailwind.config.ts` 文件，修改 `primary` 颜色配置。

**Q: 如何添加新页面？**
A: 在 `app` 目录下创建新文件夹和 `page.tsx` 文件。

**Q: 文章内容支持图片吗？**
A: 目前支持，但需要将图片放在 `public` 目录，然后在 Markdown 中使用 HTML 标签。

**Q: 如何备份网站？**
A: 将代码推送到 GitHub 就是最好的备份方式。
